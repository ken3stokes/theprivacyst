User Guide

Password Calculator

Generate strong, unique passwords for all your accounts.
  • Choose password length between 8 and 128 characters
  • Select character types: uppercase, lowercase, numbers, special characters
  • Generate secure, random passwords
  • Copy generated passwords with one click
Tip: For maximum security, use all character types and a length of at least 12 characters.

SSH Key Generator

Create secure SSH keys for remote access.
  • Choose between RSA, DSA, ECDSA, and Ed25519 key types
  • Select key length (2048 or 4096 bits for RSA)
  • Generate both public and private keys
  • Copy keys directly to clipboard
Tip: Ed25519 keys are recommended for modern systems due to their security and efficiency.

Username Generator

Create unique, memorable usernames.
  • Generate random, unique usernames
  • Copy generated usernames instantly
  • Create professional-looking identifiers
Tip: Generated usernames are designed to be memorable while maintaining uniqueness.

Email Generator

Create realistic email addresses for testing or demonstration purposes.
  • Generate realistic email addresses using common first and last names
  • Multiple email formats:
    • First.Last (e.g., john.smith42)
    • First Initial.Last (e.g., j.smith123)
    • FirstLast (e.g., johnsmith789)
    • Last.First (e.g., smith.john456)
  • Choose from popular email domains (Gmail, Outlook, ProtonMail)
  • Automatically adds realistic number suffixes
  • Copy generated emails with one click
Tip: Each generated email follows common naming conventions used by real email providers, making them perfect for testing or demonstration purposes.
