The Privacyst - Release Notes 2023.3.1

The latest release notes from The Privacyst outline exciting new features, significant enhancements, crucial bug fixes, and provide insight into known issues and upcoming additions, all aimed at improving user experience and enhancing online security.

New Features:

  • Password Calculator: Generate strong and secure passwords with customizable length and character sets.
  • SSH Key Generator: Create secure SSH keys with selectable RSA key lengths.
  • Username Generator: Easily generate random usernames with options for length and prefixes/suffixes.
  • Email Generator: Generate random email addresses with selectable domains for various purposes.
  • Dark Mode: Enjoy a dark mode for reduced eye strain and improved user experience.

Enhancements:

  • Mobile Responsiveness: Improved mobile responsiveness for a seamless user experience on a variety of devices.
  • User Guide: Added a comprehensive user guide to help users understand how to leverage security tools effectively.
  • Tooltips: Added tooltips for user assistance, providing information on various features and options.

Bug Fixes:

  • Header Display Issue: Resolved the issue with the header not displaying correctly on mobile devices.
  • Theme Persistence: Fixed a bug where selected themes were not persisting after page reloads.

Known Issues:

  • Password Strength Indicator: A visual password strength indicator will be added in the next version.
  • User Accounts: User account functionality will be introduced in the next version.
  • Download SSH Keys: Download options for SSH keys will be available in the next version.

Coming Soon:

Look forward to our next release, which will introduce advanced features like password complexity options, user accounts, and the ability to download SSH keys. We appreciate your feedback and hope you find these security tools valuable for enhancing your online security.
